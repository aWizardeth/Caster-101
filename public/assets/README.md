# Static Assets Structure

This folder is prepared for extracted static assets.

- `css/index/` — CSS extracted from `index.html`
- `css/caster-valley/` — CSS extracted from `caster-valley.html`
- `js/index/` — JS modules extracted from `index.html`
- `js/caster-valley/` — JS modules extracted from `caster-valley.html`

Current runtime remains unchanged until files are linked from HTML.
